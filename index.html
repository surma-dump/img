img

img provides a bunch of image manipulation tools. It's (nowhere near) a command line photoshop.

Usage

Each img tool expects an image from STDIN to work on, and will print the result to STDOUT, like so

$ img greyscale < input.png > output.png

Using this method it is easy to chain multiple commands together using pipes,

$ (img greyscale | img pxl | \
    img contrast --by 0.05) \
    < input.png > output.png

Tools

Below is a list of each describing each tool, showing how to use them from the command line and within go. The go examples only show code relevant to using the img function. And the example images shown are generated using the command with no options provided, they are not produced using the example code shown.

brightness

Brightness adjusts the brightness of an image. It multiplies each colour channel by the value given.

$ img brightness --by 0.1 < input.png > output.png
import "img/brightness"

func brighten(img image.Image) image.Image {
  return brightness.Adjust(img, 0.1)
}

contrast

Contrast adjusts the contrast of an image. This alters the differences between colours. Increasing contrast can make details in the image more visible, while lowering contrast can help reduce excessive sharpness.

$ img contrast --by 0.1 < input.png > output.png
import "img/contrast"

func increaseContrast(img image.Image) image.Image {
  return contrast.Adjust(img, 0.1)
}

saturation

Saturation adjusts the saturation of an image. The saturation of a colour refers to the perceived intensity of a colour. So increasing the saturation will make colours brighter and more vivid.

$ img saturation --by 0.1 < input.png > output.png
import "img/saturation"

func saturate(img image.Image) image.Image {
  return saturation.Adjust(img, 0.1)
}

hue

Hue adjusts the hue of an image. The hue can be thought of as the angle a colour is around the colour wheel; with 0° as red, 120° being green, 240° blue and then the cycle repeats with 360° being red again. This shifts the whole circle so a change of 120 would make reds become green, greens become blue and blues become red.

$ img hue --by 180 < input.png > output.png
import "img/hsla"

func spinColours(img image.Image) image.Image {
  return hsla.Hue(img, 180)
}

lightness

Lightness adjusts the lightness of an image. This is similar to brightness but whereas brightness considers the average of the sum of colour channel values, lightness considers the average of the two brightest colour channels.

$ img saturation --by 0.1 < input.png > output.png
import "img/lightness"

func lighten(img image.Image) image.Image {
  return lightness.Adjust(img, 0.1)
}

greyscale

Greyscale creates a greyscale version of the image. There are multiple methods of conversion:

  • average - uses the average of each colour channel
  • lightness - uses the average of the two brightest colour channels
  • luminosity - uses a weighted sum of the colour channels to better represent the colours humans' perceive greatest
  • maximal - uses the greatest value of each colour channel
  • minimal - uses the least value of each colour channel
  • photoshop - modified version of luminosity, (supposedly) used in Adobe Photoshop
$ img greyscale --maximal < input.png > output.png
import "img/greyscale"

func greyify(img image.Image) image.Image {
  return greyscale.Maximal(img)
}

pixelate

Pixelate creates a pixelated image. It breaks the image into rectangles of the size specified and averages the colours in each.

$ img pixelate --size 20x30 < input.png > output.png
import "pixelate"

func pixelise(img image.Image) image.Image {
  return pixelate.Pixelate(img, 20, 30)
}

pxl

Pxl pixelates an image into right-angled triangles blocks, averaging the colour in each. By default the “direction” of the triangles will be chosen based on the “closeness” of the average colour in the each quadrant.

$ img pxl --size 50x50 < input.png > output.png
import "img/pixelate"

func pxlise(img image.Image) image.Image {
  return pixelate.Pxl(img, 50, 50)
}

hxl

Hxl pixelates an image into equilateral triangles, averaging the colour in each. The triangles are arranged in a hexagonal pattern.

$ img hxl --width 30 < input.png > output.png
import "img/pixelate"

func hxlise(img image.Image) image.Image {
  return pixelate.Hxl(img, 30)
}

blend

Blend takes two images and produces a result by blending the colours in each together. There are a lot of different blend modes, here is an image showing all of them.

$ img blend --overlay other.png < input.png > output.png
import "img/blend"

func compose(base, blend image.Image) image.Image {
  return blend.Overlay(base, blend)
}

Using some imagination...

With a little imagination you can combine the tools to produce super cool effects. Use pipes in the shell, or just call multiple functions in code.

$ (img shuffle --horizontal | img hxl | \
    img hue --by -20) < input.png > output.png
import (
  "img/shuffle"
  "img/pixelate"
  "img/hsla"
)

// May require more imaginative naming...
func coolEffect(img image.Image) image.Image {
  img = shuffle.Horizontal(img)
  img = pixelate.Hxl(img)
  return hsla.Hue(img, -20)
}
